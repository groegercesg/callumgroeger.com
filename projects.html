So... What have you done?

Over my many years of doing stuff, lots of stuff has been done; clearly I'm not the most active contributor on Github or the most profilic writer.

But here's a fairly complete list of the major projects that I've worked on:

  • Movie Barcode (2019) - A Python script to produce 'barcodes' from media files, making images in a nicely formatted and presentable manner.
  • Callum Sort (2018) - An brand new sorting algorithm, worse than Bubble but better than Insertion sort (So pretty middle of the road).
  • PIX (2017-2019) - An automated script for capturing particle data with the MX-10, written in Python with use of the TAPAS api

For more information regarding any of the above projects, please do not hesitate to contact me
